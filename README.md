# OrderBot — AI-Powered Pizza Restaurant Chatbot

A Python-based conversational AI chatbot built with OpenAI's GPT-3.5-Turbo. OrderBot simulates a pizza restaurant ordering assistant that greets customers, collects their order, handles delivery/pickup details, and produces a structured JSON summary of the order — all through natural language conversation.

Built as an early exploration into prompt engineering, multi-turn chat formats, and the OpenAI Chat Completions API.

## Demo

```
OrderBot: Hey there! Welcome to our pizza restaurant! What can I get for you today?

User: Hi, I'd like a large pepperoni pizza

OrderBot: Great choice! Would you like any toppings on that? We have:
          extra cheese ($2.00), mushrooms ($1.50), sausage ($3.00),
          canadian bacon ($3.50), AI sauce ($1.50), peppers ($1.00)

User: Add extra cheese and mushrooms. And a medium coke.

OrderBot: Got it! So that's:
          - Large pepperoni pizza + extra cheese + mushrooms
          - Medium coke
          Will this be pickup or delivery?
```

**Final JSON Summary Output:**
```json
{
  "pizza": { "type": "pepperoni pizza", "size": "large" },
  "toppings": ["extra cheese", "mushrooms"],
  "drinks": [{ "type": "coke", "size": "medium" }],
  "sides": [],
  "total price": 18.45
}
```

## Features

🍕 **Automated Order Collection**
- Greets customers naturally and guides them through the menu
- Clarifies sizes and options before confirming items

🚗 **Delivery & Pickup Handling**
- Asks whether the order is pickup or delivery
- Collects delivery address when needed

💬 **Multi-Turn Conversation Memory**
- Maintains full conversation context across the session
- Remembers earlier messages to answer follow-up questions

📋 **Structured Order Summary**
- Generates a clean JSON breakdown of the order at the end
- Itemizes pizza, toppings, drinks, sides, and total price

🖥️ **Interactive Panel UI**
- Runs as an interactive chat panel inside Jupyter Notebook
- Real-time conversation with a Chat button and text input

## Tech Stack

- **UI**: Panel (Jupyter widget)

## Getting Started

### Prerequisites

- Python 3.9 or higher
- An OpenAI API key ([get one here](https://platform.openai.com/api-keys))
- Jupyter Notebook or JupyterLab

### Installation

1. Clone the repository
```bash
git clone https://github.com/your-username/orderbot.git
cd orderbot
```

2. Install dependencies
```bash
pip install openai python-dotenv panel jupyter
```

3. Create a `.env` file in the project root
```
OPENAI_API_KEY=your_openai_api_key_here
```

4. Launch Jupyter Notebook
```bash
jupyter notebook
```

5. Open `orderbot.ipynb` and run all cells

### Usage

- Run all cells in sequence
- The last cell renders an interactive chat panel
- Type your order in the text box and click **Chat!**
- The bot will guide you through the full ordering flow
- After the conversation, run the summary cell to get a JSON order breakdown

## How It Works

The bot uses OpenAI's **chat format** — a list of messages with roles (`system`, `user`, `assistant`) sent on every API call:

```python
context = [
    {'role': 'system', 'content': 'You are OrderBot... [full menu and instructions]'},
    {'role': 'user',   'content': 'I want a large pepperoni pizza'},
    {'role': 'assistant', 'content': 'Great! Any toppings?'},
    {'role': 'user',   'content': 'Extra cheese please'}
]
```

Each new message is appended to the context list and the full history is sent back to the model, giving it memory of the conversation.

The final JSON summary is generated by injecting a second `system` message into the conversation context asking the model to parse and format the order.
